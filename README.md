# booking-system
Create a simple bus booking website
To run the website use:- node server.js / npm run dev
Use this link on browser:- http://localhost:5000

### Backend
- Node.js & Express.js
- MongoDB with Mongoose
- JWT Authentication
- Socket.io for real-time features

### Frontend
- HTML5, TailwindCSS
- JavaScript (ES6+)
- Font Awesome icons
- Leaflet maps for tracking

## Add Sample Bus Data
- node add-buses.js
